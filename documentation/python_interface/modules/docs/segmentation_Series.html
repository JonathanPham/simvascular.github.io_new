get_segmentation(id, time=0) 
   
   Get the segmentation for the given id and time.                            
   
   The segmentation id identifies a segmentation from a list of segmentations 
   defined for each time point. The id is between 0 and the number of         
   segementations - 1.                                                        
   
   Args: 
     id (int): The id of the segmentation to get. 0 <= id <= #segmentations-1 
     time (Optional[int]): The time to get the segmentation for.              
   
   Returns (sv.segmentation.Segmentation): The segmentation object for the    
      given id and time.
