TetGen adaptive meshing options class functions

Methods 
 get_values()  
  
  Get the names and values of TetGen adaptive mesh generation options. 
  
  Args:  
    None

Data 
end_step
    Simulation end step (int). The last simulation step. 


max_edge_size
    Global maximum edge size (float). Specify a maximum target edge size. No edge size will be larger than  
 this size, even if the adaptor identifies that the solution does not require an edge length this small. 


min_edge_size
    Global minimum edge size (float). Specify a minimum target edge size. No edge size will be smaller than  
  this size, even if the adaptor identifies that solution needs a edge length smaller than this. 


start_step
   Simulation start step

step_increment
   Simulation step increment.

use_isotropic_meshing
    Use an isotropic meshing methods (bool). This is currently the only option. 


use_multiple_steps
   If True then use multiple simlation steps.
