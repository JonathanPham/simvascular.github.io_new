The BlendOptions class stores parameter values used to control how       
blended surfaces are generated.                                         

Example: Create a blend options object                                   

    options = sv.geometry.BlendOptions()

Methods 
get_values()  
  
  Get the names and values of blend options. 
  
  Returns (dict): A dict with key/value pairs for each option name/value.

Data 
num_blend_operations
   Type: int                                                               
   Default: 2                                                              
   
   The number of times to perform blending operations. A series of subbblend
   opertations and a subdivision operation is performed for each blending  
   operation.                                                              


num_cgsmooth_iterations
   Type: int                                                               
   Default: 2                                                              
   
   The number of conjugate gradient iterations used in the constrained     
   constrained smoothing computation performed for each subbblend operation.


num_lapsmooth_operations
   Type: int                                                               
   Default: 50                                                             
   
   The number of laplacian smoothing operations to perform for each        
   subbblend operation.                                                    


num_subblend_operations
   Type: int                                                               
   Default: 3                                                              
   
   The number of subdivision operations to be performed within each blending
   operation. Each subbblend operation performs: constrained smoothing,    
   Laplacian smoothing, and decimation operations.                         


num_subdivision_operations
   Type: int                                                               
   Default: 1                                                              
   
   The number of subdivisions operations to perform at the end of each     
   blending operation. Each triangle in the surface is divided into four   
   new triangles for each subdivision operation.                           


target_decimation
   Type: float                                                             
   Default: 1.0                                                            
   
   The percent of triangles to remove from the smoothed surface each       
   subblend operation.
